What is the use of GitHub?

Introduction. GitHub is a code hosting platform for version control and collaboration. It lets you and others work together on projects from anywhere. This tutorial teaches you GitHub essentials like repositories, branches, commits, and pull requests.

What does hub mean in GitHub?

hub is an extension to command-line git that helps you do everyday GitHub tasks without ever leaving the terminal. Read the full documentation: man hub, or visit this project on GitHub.

What is Git and why it is used? Git is a DevOps tool used for source code management. It is a free and open-source version control system used to handle small to very large projects efficiently. Git is used to tracking changes in the source code, enabling multiple developers to work together on non-linear development
Should I learn Git or GitHub? Developers normally prefer git as it requires commands to enter and work is done. For a beginner, I suggest go for GitHub to understand how it works, looks and what are different features there like pull compare, commit with description, fork etc., We get more clarity when we have a visual available.
